# 地区切换功能测试教程

## 快速测试步骤

### 1. 测试全国 vs 上海

**步骤：**
1. 打开小程序首页
2. 点击顶部地区切换按钮
3. 选择"**全国**" → 返回首页
4. 记录看到的帖子数量（应该是全部）
5. 再次点击地区切换
6. 选择"**上海**" → 返回首页
7. 观察帖子数量减少（只显示全国可见的帖子）

**预期结果：**
- 全国：显示所有帖子
- 上海：帖子数量减少，只显示全国可见的内容

---

### 2. 测试不同城市

**步骤：**
1. 依次切换：北京 → 广州 → 深圳 → 天津 → 南京
2. 每次切换后观察首页帖子变化

**预期结果：**
- 每个城市显示的帖子组合不同
- 全国可见的帖子在所有城市都能看到
- 本城市可见的帖子只在对应城市显示

---

## 帖子可见性规则

| 可见范围 | 说明 |
|---------|------|
| **全国可见** | 所有地区都能看到 |
| **本城市可见** | 只有同城用户能看到 |
| **本校区可见** | 只有同校区用户能看到 |
