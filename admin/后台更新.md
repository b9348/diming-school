## 五、新增API接口

### 中间层需新增
```
GET    /admin/banner/list           # 轮播图列表
POST   /admin/banner/saveOrUpdate   # 创建/更新/删除（传 id 更新，传 id + deleted: true 删除）

GET    /admin/nav/list              # 导航列表
POST   /admin/nav/saveOrUpdate      # 创建/更新/删除

GET    /admin/module-config/:module # 获取模块配置
POST   /admin/module-config/saveOrUpdate   # 保存模块配置

GET    /admin/forum/list            # 论坛列表
POST   /admin/forum/saveOrUpdate    # 创建/更新/删除
POST   /admin/forum/:id/admins      # 分配管理员
```

### 后台API封装
```javascript
// src/api/index.js 新增
export const bannerApi = {
  list: (params) => request.get('/admin/banner/list', { params }),
  saveOrUpdate: (data) => request.post('/admin/banner/saveOrUpdate', data)
}

export const navApi = {
  list: (params) => request.get('/admin/nav/list', { params }),
  saveOrUpdate: (data) => request.post('/admin/nav/saveOrUpdate', data)
}

export const forumApi = {
  list: (params) => request.get('/admin/forum/list', { params }),
  saveOrUpdate: (data) => request.post('/admin/forum/saveOrUpdate', data)
}
```
